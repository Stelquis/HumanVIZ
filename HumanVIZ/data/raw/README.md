### dataSet/ — 结构化的 JSON 剧本数据

与 `dataPDF/` 一一对应，将 PDF 中的内容提取为结构化 JSON，共 **1,473 个文件**。每个 JSON 文件包含以下字段：

| 字段 | 说明 |
|------|------|
| `source_folder` | 来源目录编号 |
| `source_folder_name` | 来源丛书/名家名称 |
| `file_name` | 原始 PDF 文件名 |
| `剧本名字` | 剧名（含又名） |
| `主要角色` | 角色列表及行当 |
| `情节` | 剧情简介 |
| `注释` | 注释说明 |
| `说明` | 整理说明 |
| `正文对话` | 完整剧本对话（含唱词、念白、动作提示） |
