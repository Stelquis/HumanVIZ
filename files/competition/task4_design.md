### 1.4 剧本正文结构

京剧剧本正文具备天然的结构化标记，为叙事结构分析提供了精确的解析基础：

```
【第一场】              ← 场景边界
（诸葛亮上。）           ← 舞台指示
诸葛亮 （念） 兵扎祁山地...  ← 角色 + 表演类型 + 台词
旗牌 （白） 来此已是...
诸葛亮 （西皮摇板） ...    ← 具体唱腔板式
```

**关键信号：**
- **场景标记**：`【第X场】` / `【第X场：标题】` / `【第X折】` — 天然的结构边界
- **表演类型**：`（唱）/（念）/（白）/（做）/（打）` — 叙事节奏的直接指标
- **唱腔板式**：`西皮摇板/西皮慢板/西皮二六/西皮导板/二黄原板...` — 情绪强度的信号（快板→紧张，慢板→抒情）
- **舞台指示**：`（笑）/（哭）/（急急风）/（四击头）` — 剧情情绪标记

## 二、Phase 1: 批量结构指纹提取（1473 本全覆盖）

### 2.1 目标

用纯正则方法，对所有 1473 本提取统一的结构特征向量，无需 LLM 参与。

### 2.2 提取的特征维度（约 30 维）

| 维度类别 | 具体特征 | 叙事结构含义 |
|---------|---------|------------|
| **场景规模** | 场次数、总行数、平均每场行数、场景行数变异系数 | 叙事复杂度与节奏均匀度 |
| **表演类型分布** | 唱类占比、念类占比、白类占比、做/打类占比、同场表演占比 | 叙事模式（唱工戏/做工戏/文武兼备） |
| **唱腔细分** | 西皮类占比、二黄类占比、板式多样性指数、快板/慢板比 | 情绪表达丰富度与张力 |
| **场景节奏** | 场景间行数变化率、首末场长度比、最长场位置(归一化) | 节奏起伏的幅度与高潮位置 |
| **情绪标记** | 笑/哭出现频次、冲突类舞台指示密度、急急风等锣鼓经密度 | 情绪基调（喜剧/正剧/悲剧倾向） |
| **角色密度** | 总角色数、平均每场角色数、主角对话集中度(Top-3占比) | 叙事聚焦类型（群像/单人） |

### 2.3 实现方式

```python
# 对每本剧本的"正文对话"字段执行正则提取:
#
# 1. 场景分割
scenes = re.findall(r'【第?[^】]*[场折幕]】|【[^】]*::[^】]*】', dialogue)
#
# 2. 表演类型分类
performance_markers = re.findall(
    r'（([^）]*(?:唱|念|白|做|打|板|调|腔|引|哭|笑)[^）]*)）',
    dialogue
)
#
# 3. 舞台指示提取
stage_directions = re.findall(
    r'（[^）]*(?:上|下|过场|起鼓|急急风|四击头|乱锤)[^）]*）',
    dialogue
)
#
# 4. 角色识别
characters = re.findall(r'^(\S+)\s+（', dialogue, re.MULTILINE)
```

### 2.4 产出

`structural_fingerprints.json` — 1473 条 × ~30 维特征矩阵

## 四、Phase 3: 典型剧本选取策略

### 4.1 分层抽样逻辑

```
Step 1: 构建 7×5 分层网格 (剧目类型 × 来源大类)
        → 35 个格单元
        → 标注每格的剧本数量
        → 标记空单元格

Step 2: 对每个非空单元格:
        a. 计算格内所有剧本的结构特征均值向量（重心）
        b. 对格内剧本按叙事结构聚类 (k = min(3, 格内样本数))
        c. 每个子类选离聚类中心最近的 1 本 → "典型代表"
        d. 同时选离中心最远的 1 本 → "边缘/独特样本"

Step 3: 特殊覆盖规则:
        - 无场景标记型: 至少选 3 本（占 21.5%，不能忽略）
        - 昆曲来源: 至少选 3 本（唯一非京剧，结构有独特意义）
        - 有次要剧目类型的 744 本: 确保跨类型标签被覆盖
        - 技法展示戏: 全部覆盖（仅 17 本）
        - 每家名家至少 1 本（13 位名家，需覆盖流派结构差异）

Step 4: 校验与调整:
        - 校验: 7 剧目类型 ✓ | 5 来源大类 ✓ | 5 结构类型 ✓
        - 预估选出: 40-60 本典型剧本
```

## 五、Phase 4: 深度叙事分析（40-60 典型本）

### 5.1 Layer 1 — 场景级节奏标注（规则 + LLM 混合）

对每场标注：

| 标注维度 | 提取方法 | 取值 |
|---------|---------|------|
| **表演形式配比** | 正则统计唱/念/做/打/白行数占比 | 5 维百分比向量 |
| **唱腔板式序列** | 正则提取西皮/二黄 × 板式类型 | 序列标签 |
| **情绪强度** | 唱腔板式映射（快板=高，慢板=中）+ LLM 打分 | 1-10 分 |
| **冲突级别** | 打斗标记密度 + 急急风等锣鼓经 + LLM 打分 | 1-10 分 |
| **叙事功能** | LLM 判断：起/承/转/合 | 四分类标签 |
| **出场角色** | 正则提取 + 新角色/延续角色标记 | 角色列表 |

### 5.2 Layer 2 — 全局叙事模式识别

对每本剧本：

| 分析维度 | 方法 | 产出 |
|---------|------|------|
| **叙事弧线** | 场景 × 情绪强度 曲线拟合并分类 | 峰型 / 坡型 / 谷型 / 双峰型 / 平缓型 |
| **阶段边界** | 起→承、承→转、转→合的转折场景位置 | 4 阶段占比 |
| **节奏模式** | 场景间情绪差值的自相关分析 | 紧-松-紧 / 松-紧-松 / 匀速递进 / 爆发式 |
| **收束方式** | 末场景情绪值与表演类型判断 | 团圆式 / 开放式 / 悲壮式 / 教化式 |

### 5.3 Layer 3 — 表演形式-叙事功能映射

京剧表演形式的叙事功能分析框架：

| 表演形式 | 叙事功能 | 典型场景 |
|---------|---------|---------|
| **唱 (慢板/原板)** | 抒情独白、内心活动外化、情感渲染 | 月下思人、狱中诉冤 |
| **唱 (快板/摇板)** | 情绪爆发、冲突升级、急迫叙事 | 闻报惊变、怒斥奸佞 |
| **唱 (流水板)** | 快速叙事交代、信息传递 | 路程行进、事件复述 |
| **念 (引子/定场诗)** | 角色登场亮相、身份建立 | 每场开场 |
| **念 (宣读)** | 关键情节推动（圣旨/书信/状纸） | 公堂、军帐 |
| **白** | 日常对话、喜剧互动、剧情推演 | 贯穿全剧 |
| **做** | 无台词表演推进、内心外化 | 哑剧段落 |
| **打** | 冲突高潮、正邪对决、武力展示 | 战场、搏斗 |

## 六、Phase 5: 可视化设计（改造 Story Ribbons）

### 6.1 编码改造

#### 原始 Story Ribbons 编码 → 京剧叙事 Ribbon 编码

```
原始 Story Ribbons:
  Ribbon 条  = 角色轨迹
  Y 轴      = 角色名称
  颜色      = 角色情绪 (连续色阶)
  粗细      = 角色重要性

京剧叙事 Ribbon (每本剧本一条主 Ribbon):
  X 轴      = 场景序列 (归一化为 0%-100% 进度)
  Y 轴      = 三层叠加:
              Layer A (上): 情绪强度曲线 (红白渐变)

              Layer C (下): 冲突强度热力 (蓝白渐变)
  宽度      = 对话密度 / 多角色同场程度
  背景色带  = 起承转合四阶段 (4 色半透明)
  子 Ribbons = 每个主要角色的出场/唱段密度
```

### 7.2 Phase 4 核心改造点

## 八、与 Story Ribbons 论文的差异化

| Story Ribbons 论文 | 本方案创新点 |
|-------------------|------------|
| 处理英文小说 | 处理京剧剧本（文言 + 板腔体 + 角色行系统） |
| 角色/主题为 Ribbon 编码核心 | **表演形式 + 情绪强度** 为 Ribbon 编码核心 |
| LLM 推断场景边界（地点变化） | **正则提取显式场景标记**，LLM 补充语义标注 |
| 情感/冲突/重要性三维分析 | 新增 **唱念做打比例、板式节奏、起承转合阶段** |
| 单作品深度分析为主 | **横向多作品并排对比** + 聚类探索 |
| 通用文学分析 | **京剧叙事学专用**：行当、板式、流派作为分析维度 |
| 角色间关系网络 | 新增 **表演形式-叙事功能映射** 分析框架 |
